## üöÄ Getting Started

### Prerequisites

- **Git** - [Download & Install Git](https://git-scm.com/downloads)
- **VS Code** - [Download VS Code](https://code.visualstudio.com/) (recommended)
- **Live Server Extension** - For local development server

### üîß Local Development Setup

#### Option 1: Using VS Code (Recommended)

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/kravi-analytics-landing.git
   cd kravi-analytics-landing
   ```

2. **Open in VS Code**
   ```bash
   code .
   ```
   *Or open VS Code and use `File > Open Folder` to select the project directory*

3. **Install Live Server Extension**
   - Open VS Code Extensions panel (`Ctrl+Shift+X` / `Cmd+Shift+X`)
   - Search for "Live Server" by Ritwick Dey
   - Click "Install"

4. **Launch the Development Server**
   - Right-click on `index.html` in VS Code
   - Select "Open with Live Server"
   - Your browser will automatically open to `http://localhost:5500`


#### Option 2: Direct File Opening

1. **Download ZIP**
   - Go to the GitHub repository
   - Click "Code" > "Download ZIP"
   - Extract the ZIP file

2. **Open Locally**
   - Navigate to the extracted folder
   - Double-click `index.html` to open in your default browser
   - *Note: Some features may not work without a local server*

### üåê Live Development Server

**Why use a local server?**
- Enables proper JavaScript module loading
- Allows testing of responsive design
- Simulates production environment
- Enables hot reload for faster development

**Alternative local servers:**
```bash
# Using Python (if installed)
python -m http.server 8000

# Using Node.js (if installed)
npx serve .

# Using PHP (if installed)
php -S localhost:8000
```
